### ⚙️ What It Targets

Currently, StateStinger focuses on:

- ✅ **Cosmos SDK apps**
  - `DeliverTx`, `BeginBlock`, `EndBlock`
  - Built-in modules like `x/bank`, `x/staking`, `x/auth`

Planned support:
- ☑️ Ethermint
- ☑️ Tendermint Core fuzzing
- ☑️ Other Go-based blockchains (e.g., Celestia Core, Hyperledger Fabric)
